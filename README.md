# Data Science Portfolio

This repository contains practical projects from a Data Science learning path.

It demonstrates end-to-end workflow from data processing to model training and basic deployment.

---

## 01 Data Pipeline
Data preparation and analysis:
- data cleaning
- missing value handling
- exploratory data analysis (EDA)
- feature engineering

---

## 02 Machine Learning Pipeline
Classic ML workflow:
- model training (sklearn)
- linear models
- model evaluation
- comparison of approaches

---

## 03 MLOps Pipeline
Basic production-oriented practice:
- model deployment (FastAPI)
- API service structure
- Docker-based deployment
- simple monitoring and orchestration concepts

---

## Tech Stack
Python, pandas, numpy, scikit-learn, FastAPI, Docker, Airflow, Grafana
